## 📖 项目概述

本项目致力于**新能源汽车电池的故障检测与早期预警**。系统集成了 **5 种不同类型的电池故障识别算法**，并结合 **1 种基于时间序列的高级预测算法**，能够多维度地分析电池运行数据，提前防范潜在的安全风险。

---

## 📂 核心模块说明

项目代码结构清晰，主要分为**故障检测**与**故障预警预测**两大部分：

### 🛠️ 故障检测模块
| 文件名 | 功能描述 | 核心算法 / 判定方法 |
| :--- | :--- | :--- |
| `Connection_Fault.py` | **连接异常检测** | 基于 ICC（组内相关系数） |
| `Failure_of_Insulation.py` | **绝缘失效检测** | 基于充电状态的动态阈值 |

| `Sudden_internal_Fault.py` | **突发内部短路检测** | V/T 综合分析 + SDO 谱密度异常算法 |

### 📈 故障预警模块

## ✨ 功能特性详解

### 1. 🔌 连接故障检测 (`Connection_Fault.py`)
* **应用场景**：检测电池单元（Cell）之间是否存在连接松动或阻抗异常。
* **技术原理**：通过分析相邻电压序列的相关性来评估连接健康度。
* **判定标准**：使用 **ICC（组内相关系数）** 进行计算，当 ICC 值 **< 0.8** 时，系统将自动标记为连接异常。

### 2. 🛡️ 绝缘失效故障检测 (`Failure_of_Insulation.py`)
* **应用场景**：实时监测电池包绝缘电阻，防止漏电及高压风险。
* **技术原理**：系统会根据电池当前所处的**充电状态（State 1-4）**，自适应设定不同的绝缘电阻安全阈值。
* **判定标准**：动态识别绝缘失效模式，有效减少误报。

### 3. 📊 采样故障检测 (`Sampling_Fault.py`)
* **应用场景**：甄别 BMS 数据采集过程中因传感器或通信问题产生的异常跳变。
* **技术原理**：利用**滑动窗口 (Sliding Window)** 实时扫描电压数据流。
* **判定标准**：基于**分位数方法**，自动过滤并标记出异常的采样噪点。

### 4. 🔋 自放电故障检测 (`Self_Discharge_Fault.py`)
* **应用场景**：发现电池电芯内部微短路或异常掉电导致的自放电率过高问题。
* **技术原理**：采用**滚动窗口平均方法**平滑电压变化曲线，消除短期波动干扰。
* **判定标准**：引入 **IQR（四分位距）** 统计学方法，精准识别电压下降速率的异常特征点。

### 5. ⚠️ 突发内部短路故障检测 (`Sudden_internal_Fault.py`)
* **应用场景**：防范极具破坏性的电池突发性热失控及内部短路。
* **技术原理**：深度融合**电压**与**温度**双重维度数据进行综合状态分析。
* **判定标准**：应用 **SDO（谱密度异常）算法**，快速捕捉短路前兆，实现毫秒级故障检测。

### 6. 🔮 故障预测算法 (`TCN-GAWO.py`)
* **应用场景**：基于历史时序数据对电池的未来健康状态进行超前预警。
* **技术原理**：采用高级时间序列预测模型，辅助特征优化算法，挖掘数据潜在规律，将“被动检测”升级为“主动预警”。
